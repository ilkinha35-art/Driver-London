# Driver Arcade

Um jogo estilo arcade inspirado em Driver: London, feito em **HTML, CSS e JavaScript**, sem necessidade de APK ou engine.

## 🎮 Como jogar
- Use as setas do teclado (ou controles touch no celular).
- Desvie dos obstáculos e tente sobreviver o máximo de tempo.
